# Pigmon - Gestão Financeira

Uma interface web moderna e responsiva focada na gestão financeira pessoal. O projeto propõe um fluxo de navegação intuitivo, permitindo que o usuário acesse desde a sua autenticação até painéis detalhados de métricas, controle de margem disponível e organização mensal de despesas através de um calendário estruturado.

### Calendário Financeiro
Esta interface permite a visualização e organização estruturada de gastos, contas e recebimentos ao longo dos dias do mês selecionado.

* **Visão Mensal:** Grade de calendário para acompanhar os dias úteis e os compromissos financeiros de forma visual e direta.
* **Categorização por Cores:** Identificação visual do status de cada lançamento financeiro diretamente nos dias:
  * Pendente: Contas a pagar sinalizadas com a cor vermelha (ex: parcelas, assinaturas de streaming).
  * Pago: Contas já quitadas e validadas sinalizadas em verde.
  * Avulso: Gastos esporádicos ou não recorrentes sinalizados em azul.
* **Ações Rápidas:** Botões dedicados para inserção ágil de dados no sistema, como "Registrar gasto" e "Adicionar conta".
* **Navegação Responsiva:** Barra lateral de menu que se adapta perfeitamente ao formato de bottom-bar (barra inferior) em dispositivos móveis, garantindo a usabilidade em qualquer tamanho de tela.

## Tecnologias Utilizadas

O desenvolvimento do front-end foi realizado utilizando as seguintes tecnologias:

* HTML5 para a estruturação semântica.
* CSS3 puro para estilos e proporções específicas.
* Tailwind CSS para a construção rápida do layout e garantia da responsividade (abordagem Mobile First).
* FontAwesome para a renderização de iconografia na barra de navegação e botões.
* Google Fonts para a tipografia padrão do projeto (família Space Grotesk).
